How we rank vanilla Hytale servers

This list is curated manually. There are no votes and no automated ranking. Each server is reviewed against the same rubric to keep the “vanilla” label meaningful and comparable.

Exclusion rules (must pass)

  • No pay-to-win: no progression advantages sold for money.
  • Vanilla must be defined: the server must clearly state what is changed and what is not.
  • Rules must be public: griefing, PvP, resets, and moderation expectations.
  • Contact must exist: at least an active Discord or website.
  • Verifiable access: IP and availability must be checkable (we note “last checked”).

“Vanilla” here means: base gameplay first, minimal additions, and full disclosure. QoL may exist, but it should not rewrite progression.

Editor score (0 to 100)

We score each server using weighted criteria, then rank by the final score.

  • 35 Vanilla Purity
  • 20 Fairness and No P2W
  • 15 Stability and Last Checked
  • 15 Rules and Moderation
  • 10 Community and Longevity
  • 5 Transparency

We keep a “Last checked” date on every listing. If a server changes policy or additions, its score can change on the next review.

Tip: if you run a vanilla Hytale server and want to be listed, make your vanilla definition, rules, and monetization policy easy to find.

How to join a vanilla Hytale server

The exact menu labels may change over time, but the basic flow stays the same.

  1. Open Hytale.
  2. Go to Multiplayer (or Servers).
  3. Choose “Direct Connect” (or “Add Server”).
  4. Paste the IP from the card above and join.

Some vanilla servers use whitelist onboarding. If a server is whitelisted, join their Discord and follow the “start here” steps.

Vanilla Hytale server FAQ

Clear definitions help keep the list useful. These FAQs also cover common questions people search for.

What is a vanilla Hytale server? Definition

A vanilla Hytale server focuses on base gameplay. It avoids additions that significantly change progression, combat balance, loot, or economy. Some servers may still run minimal QoL and moderation tools, but those should be disclosed.

Vanilla vs semi-vanilla: what is the difference? Comparison

Semi-vanilla usually allows a few convenience features (for example, limited teleports, basic claims, or UI helpers). In this list, we prefer minimal additions and we document them per server.

Do these servers allow mods or custom content? Content

Some servers may run server-side additions. We require clear disclosure and we score how strongly they impact “vanilla purity”.

Are these servers pay-to-win? Fair play

Our list excludes pay-to-win servers. We review monetization policies and document what is sold. If a server changes its store policy, it can be removed.

How often is this list updated? Updates

This is a small curated page. We update when we re-check listings or when a server policy changes. The “Last updated” date appears at the top.

Can I suggest a server? Submissions

Yes. Send the server name, IP, region, language, and links to rules and monetization policy. If it passes the exclusion rules, we may review it.
